面向对象:万事万物皆对象

    类:
        把对象的属性(变量)和行为(方法)抽象出来
    对象:
        把属性具体化,实例化

总述:
    类是对一类事物的描述,是抽象的
    对象是一类事物的实例,是具体的
    类是对象的模板,对象是类的实体

我要抽象出一个类:cat
    属性:age,colar,weight,name
    方法:run(),jump().sleep()


java中用class描述事物:
    成员变量:事物的属性
    成员方法:事物的行为
类中成员变量的默认值:
    基本类型:
         字符(char)                                '\u0000'
         布尔型(关键)                               false
         整数型(关键)                               0            
         浮点型(关键)                               0.0
    引用类型:
         数组,类,接口(关键)                         null
