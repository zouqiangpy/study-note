四种权限修饰符

public 公共的
    都可以对其进行访问

protected 受保护的
    不同包子类和本包的类能够访问的成员变量和方法

default 默认的
    只有同一个包下可以被访问
    注意:普通变量不加修饰符就是这个默认修饰符

private 私有的
    只有本类中能够访问

               public    protected    default    private

本类            是          是         是          是

本包所有类      是          是         是     

不同包子类      是          是         

不同包无关类    是


编写代码时,如果没有特殊要求,建议这样使用修饰符.

成员变量    private 只供本类使用,隐藏细节
成员方法    public 方便调用方法
构造方法    public 方便创建对象
